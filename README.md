# Sambit Hota - Portfolio Website

A **modern, responsive, and animated portfolio website** showcasing my skills, projects, and education.  
Built using **HTML** and **CSS** with a professional design, smooth animations, and a clean layout.

## 📌 Features
- **Responsive Design** – Works on mobile, tablet, and desktop.
- **Modern Color Palette** – Gradient blue-purple theme.
- **Smooth Animations** – Fade-in, hover effects, and interactive cards.
- **Profile Photo Integration** – Displays a professional circular profile image.
- **Well-Organized Sections**:
  - Summary
  - Technical Skills
  - Projects
  - Education
  - Contact Information

## 🛠 Tech Stack
- **HTML5**
- **CSS3**
- **Google Fonts (Poppins)**
- **Responsive Grid Layout**
- **CSS Animations**
